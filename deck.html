Case Study

How an AI agent built a production-ready landing page in one session.

From blank canvas to live prototype — the full process of building GrowthX's new landing page with an AI coding agent in Cursor.

The Brief

It started with a conversation.

The VP of GTM wanted a landing page that treated the user experience holistically — the before, during, and after of encountering video content. Not just a page. A system.

The inputs:

  • A reference site — storyarb.com for UX excellence
  • The brand — growthx.ai for positioning and visual identity
  • A video transcript — the CEO's pitch for the page
  • Brand guidelines — colors, fonts, tone of voice

"Make this the very best possible experience for a marketing operator."

The Process

Seven phases. One session.

Phase 1

Research

Navigated both reference sites in a real browser. Took screenshots at every scroll depth. Extracted copy, layout patterns, and SVG logos.

Phase 2

Architecture

Designed a 10-section page flow based on storyarb's proven narrative arc, adapted for GrowthX's positioning.

Phase 3

Build v1

Wrote a complete single-file HTML/CSS/JS prototype with scroll animations, responsive grid, and video embed.

Phase 4

Brand Alignment

Applied brand guidelines — Messina Sans fonts, color palette with exact hex values, logo integration, tone of voice.

Phase 5

UX Audit

Performed a 10-point UX comparison against storyarb. Identified gaps in hero impact, testimonial weight, and conversion paths.

Phase 6

Build v2

Rebuilt from scratch incorporating all audit findings — dark hero, testimonial carousel, timeline pains, newsletter capture, brand colors.

Phase 7

Ship

Created a standalone GitHub repo, compressed a video preview loop, enabled GitHub Pages. Live URL in minutes.

Capabilities Demonstrated

What the AI agent actually did.

  1. 1
    Browsed live websites — navigated storyarb.com and growthx.ai, scrolled through sections, took screenshots, and extracted inline SVG logos via JavaScript injection
  2. 2
    Interpreted visual brand guidelines — read PNG images of color palettes, typography specs, and tone-of-voice docs, then mapped them to CSS variables
  3. 3
    Wrote production-quality code — 1,500 lines of semantic HTML, responsive CSS with design tokens, and vanilla JS for scroll animations, video control, and a testimonial carousel
  4. 4
    Performed a UX audit — analyzed storyarb section-by-section as a "world-class designer" and produced a prioritized 10-point improvement plan
  5. 5
    Compressed video with ffmpeg — installed ffmpeg, trimmed a 74MB source video to a 333KB muted preview loop for the hero
  6. 6
    Deployed to production — created a GitHub repo, pushed files, enabled GitHub Pages via API, and delivered a shareable live URL

The Takeaway

This is what AI agents actually unlock.

Not "AI-generated content." Not chatbot summaries. A tireless collaborator that researches, designs, codes, audits, iterates, and ships — in a single continuous session.

  • Speed without sacrifice. Two full page versions, a UX audit, brand alignment, and deployment — in one sitting.
  • Full-stack capability. The same agent browsed websites, wrote CSS, compressed video, created git repos, and called GitHub APIs.
  • Human-directed, AI-executed. Every decision was guided by a human. The agent handled the execution at a pace no team could match.
  • Compounding context. Each iteration built on the last. No re-briefing. No context loss between handoffs.
